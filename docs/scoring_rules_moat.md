# 护城河分析评分规则（Module 3）

**总分：30 分 = 定量 5 分 + 定性 25 分**

| 维度 | 类型 | 分值 | 数据来源 |
|------|------|------|----------|
| 毛利率绝对值 | 定量 | 2.5 | SQLite `financial_reports.gross_margin` |
| 毛利率稳定性 | 定量 | 2.5 | SQLite `financial_reports.gross_margin` |
| 行业质量 | 定性(Coze) | 5 | Coze LLM |
| 护城河类型 | 定性(Coze) | 7 | Coze LLM |
| 护城河可持续性 | 定性(Coze) | 7 | Coze LLM |
| 定价权 | 定性(Coze) | 6 | Coze LLM |

---

## 1. 毛利率绝对值（2.5 分）—— 定量

计算指标：近5年平均毛利率

| 平均毛利率 | 得分 |
|------------|------|
| >= 50% | 2.5 |
| >= 40% | 2.0 |
| >= 30% | 1.5 |
| >= 20% | 1.0 |
| < 20% | 0.0 |

**数据流**：`SQLite financial_reports -> gross_margin 列最近5年 -> 取平均 -> 查表得分`

---

## 2. 毛利率稳定性（2.5 分）—— 定量

### 2.1 基础分（变异系数 CV）

近5年毛利率数据的**变异系数（CV = 标准差 / 均值）**：

> 相比绝对标准差，CV 消除了不同毛利率水平带来的量纲差异，更能反映真实的相对波动性。

| CV 范围 | 稳定性 | 基础分 |
|---------|--------|--------|
| <= 3% | 极稳定 | 2.5 |
| <= 5% | 很稳定 | 2.0 |
| <= 7% | 高度稳定 | 1.5 |
| <= 9% | 较稳定 | 1.0 |
| <= 11% | 一般 | 0.5 |
| > 11% | 波动较大 | 0.0 |

### 2.2 趋势调整

趋势差值 = 后2年平均 - 前2年平均

| 趋势方向 | 调整 |
|----------|------|
| 明显上升（>= 5%） | +1.0 |
| 温和上升（2%~5%） | +0.5 |
| 基本稳定（-2%~2%） | 0 |
| 温和下降（-5%~-2%） | -0.5 |
| 明显下降（< -5%） | -1.0 |

### 2.3 最终分

**final_score** = round(max(0, min(2.5, 基础分 + 趋势调整)) * 2) / 2

（结果以 0.5 为步长）

**数据流**：`SQLite financial_reports -> gross_margin 列最近5年 -> compute_gross_margin_score()`

---

## 3. 行业质量（5 分）—— Coze LLM

评估因素：
- 行业集中度（CR3/CR5）
- 行业成长性（近5年CAGR）
- 进入壁垒（技术/资金/牌照/品牌）
- 需求稳定性（周期性 vs 刚需）

锚点：
- 5分：极高质量行业（如高端白酒、创新药）
- 3-4分：高质量行业
- 1-2分：中等质量行业
- 0分：低质量行业（过度竞争、强周期）

---

## 4. 护城河类型与强度（7 分）—— Coze LLM

识别并评估护城河类型：
- 品牌护城河（品牌溢价、消费者认知）
- 转换成本护城河（客户粘性）
- 网络效应护城河（用户规模效应）
- 成本优势护城河（规模效应、成本领先）
- 技术优势护城河（专利、研发壁垒）
- 渠道优势护城河（终端覆盖）
- 资源垄断护城河（稀缺资源、牌照）

锚点：
- 7分：极强护城河，无法复制（如茅台品牌）
- 5-6分：强护城河，难以突破
- 3-4分：中等护城河，可能被挑战
- 1-2分：弱护城河
- 0分：无护城河

---

## 5. 护城河可持续性（7 分）—— Coze LLM + 公式计算

LLM 返回结构化字段：
- `history_duration_years`: 历史时长（年）
- `cycle_tests_count`: 周期考验轮数
- `breakthrough_difficulty`: 突破难度等级（极难4/很难3/一般2/容易1）
- `trend_judgment`: 趋势判断等级（加强3/稳定2/削弱1）

### 计算公式（满分7分，步长0.5）

```
基础分 = 历史时长
  >50年     → 5.0
  30-50年   → 4.0
  20-30年   → 3.0
  10-20年   → 2.0
  5-10年    → 1.0
  <5年      → 0.5

周期考验调整:
  ≥3轮  → +1.0
  2轮   → +0.5
  <2轮  →  0.0

突破难度调整:
  极难(4) → +1.0
  很难(3) → +0.5
  一般(2) →  0.0
  容易(1) → -0.5

趋势判断调整:
  加强(3) → +0.5
  稳定(2) →  0.0
  削弱(1) → -0.5

总分 = round(max(0, min(7, 基础分+周期+难度+趋势)) × 2) / 2
```

## 6. 定价权（6 分）—— Coze LLM + 公式计算

LLM 返回结构化字段：
- `pricing_ability`: 提价能力等级（强5/较强4/一般3/弱2/无法提价1）
- `product_uniqueness`: 产品独特性等级（稀缺4/有差异化3/一般2/同质化严重1）
- `customer_stickiness`: 客户粘性等级（极高5/高4/一般3/较低2/无1）
- `price_sensitivity`: 价格敏感度等级（高3/中2/低1）

### 计算公式（满分6分，步长0.5）

```
提价能力得分:
  强(5)      → 2.5
  较强(4)    → 2.0
  一般(3)    → 1.5
  弱(2)      → 1.0
  无法提价(1) → 0.5
  公式: level × 0.5

产品独特性得分:
  稀缺(4)        → 1.5
  有差异化(3)    → 1.0
  一般(2)        → 0.5
  同质化严重(1)   → 0.0
  公式: (level - 1) × 0.5

客户粘性得分:
  极高(5)  → 1.5
  高(4)    → 1.0
  一般(3)  → 0.5
  较低(2)  → 0.0
  无(1)    → 0.0
  公式: max(0, (level - 2) × 0.5)

价格敏感度调整（反转，敏感度越低定价权越强）:
  低(1)   → +0.5
  中(2)   →  0.0
  高(3)   → -0.5
  公式: (2 - level) × 0.5

总分 = round(max(0, min(6, 提价+独特性+粘性+敏感度)) × 2) / 2
```
